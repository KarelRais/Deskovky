Jedná se o aplikaci určenou pro evidenci deskových her.
Aplikace byla vytvořena jako projekt v rámci hodiny Programování na Obchodní akademii, Vyšší odborné škole a Jazykové škole s právem státní jazykové zkoušky Uherské Hradiště.
Autor: Karel Rais

Jak aplikaci používat:
– Přečtěte si soubory README.txt a LICENSE.txt.
– Otevřete soubor out\production\Deskovky\MainForm.class prostřednictvím Java Virtual Machine.
– Pro listování mezi hrami používejte tlačítka Předchozí/Další.
– Pro uložení parametrů použijte tlačítko Uložit.
